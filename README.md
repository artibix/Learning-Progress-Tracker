
## 打卡方法

1、创建自己的模板

2、编写check-in.md

## 打卡规则

每一个月立一个flag，记录这个月需要完成哪些内容。

每个星期立一个flag，记录这个星期需要完成哪些内容。

## 模板

```
## 2024-08-05 11:06:02 星期一

### 计划

- [x] 创建一个学习项目
- [ ] 阅读 Laravel 源码

###完成度
- [x] 创建一个学习项目...................80%
- [ ] 阅读 Laravel 源码.................90%
### 总结
-:
```
